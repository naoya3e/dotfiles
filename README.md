## Bootstrap

```sh
bash -c "$(curl -L https://raw.githubusercontent.com/naoya3e/dotfiles/master/scripts/bootstrap.sh)"
```

## Apply updates

```sh
cd $HOME/workspace/ghq/github.com/naoya3e/dotfiles
git pull --rebase
./scripts/deploy.sh
```
