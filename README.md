# .dotfiles

A dotfiles management repository

## 更新を反映する

ターミナルで以下のコマンドを入力する

```sh
$ git pull
$ sh ./install.sh
```

`install.sh`はシンボリックリンクを張るスクリプトである。
インタラクティブに確認を行うようにしているため、`replace?`とか聞かれたら`yes`と入力してシンボリックリンクを上書きしてやることでたぶん大丈夫
